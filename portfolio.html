I'm currently working on a personal book tracker app. I wanted a place I could share my thoughts about the books I read, while also keeping it personal.

I'm planning on three different versions of the site. The first version will be built with Node.js and jQuery, the second with Node.js and React, and the third with Django and React. Additionally, I would like to add additional functionality with each pass -- increased search features, user validation, etc.

I've finished with the first pass -- users have the ability to add and update books. I'm currently working on refactoring into React, including incorporating feedback from a recent code review from GitHub user mintii.
